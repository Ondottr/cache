# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| Latest  | Yes       |

## Reporting a Vulnerability

**Do not open a public GitHub issue for security vulnerabilities.**

Please report security issues by emailing **eklerk397@gmail.com** with the subject line `[SECURITY] php-sf-cache`.

Include as much of the following as possible:

- A description of the vulnerability and its potential impact
- Steps to reproduce or a proof-of-concept
- Affected versions
- Any suggested fix, if you have one

You will receive an acknowledgement within **72 hours**. I aim to release a fix within **14 days** for confirmed critical issues.

Once a fix is released I will publicly disclose the issue, crediting you unless you prefer to remain anonymous.
